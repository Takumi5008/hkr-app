# プロダクト要求定義書 (Product Requirements Document)

## プロダクト概要

### 名称
**インフラ管理** — 営業チーム向け開通管理・HKR追跡アプリ

### URL
https://infra-kanri.vercel.app

### プロダクトビジョン
営業メンバーが日々の開通実績・行動数値・チーム状況をスマートフォン・PCどちらからでも即座に確認・入力できる、チーム運営の中心ツール。

---

## ターゲットユーザー

| ロール | 説明 |
|--------|------|
| member | 一般営業メンバー。自分のデータを入力・閲覧 |
| manager | マネージャー。全メンバーのデータを閲覧・管理 |
| admin | アプリ管理者。全権限 |
| viewer | 閲覧のみのマネージャー相当 |
| shift_viewer | シフト管理専用 |

---

## 主要機能一覧

### 1. HKR入力（/input）
- So-net・WiMAX の解除数・開通数を月次で入力
- 開通表（activation_records）からの自動集計・反映
- HKR（開通率）= 開通数 ÷ 解除数 × 100、目標 80%
- 開通カレンダー（業務月単位で個人の開通予定を可視化）
- 回線管理（マネージャーのみ）

### 2. 開通表（/activation）
- So-net・WiMAX直せち・WiMAX後送り の案件を月次で管理
- 日付自動計算：獲得日→1週間後、工事日↔工事日前日、受取日↔配送前日・1週間後
- 開通カレンダーへの自動同期

### 3. ダッシュボード（/dashboard）
- 当月のHKR・解除数・開通数のサマリー
- 最近の開通フィード
- 今日のフォローアラート
- チャレンジ・スゴロクの進捗

### 4. 個人進捗（/progress）
- 月次の解除目標・実績・HKR推移グラフ
- 稼働日数・労働時間の入力

### 5. 行動表（/activity）
- 日次の行動数値入力（ピン数・ピンポン・インターホン・顔合わせ・WiMAX・So-net・解除数など）

### 6. チャレンジ（/challenge）
- チーム・個人のゲーミフィケーション目標
- ポイント・バッジ・ストリーク

### 7. 開通双六（/sugoroku）
- 開通数に連動したスゴロクゲーム

### 8. 実績管理（/performance）— マネージャーのみ
- 全メンバーの実績一覧・目標設定

### 9. チーム全体（/team）— マネージャーのみ
- チームKPIダッシュボード

### 10. その他
- 出欠管理 (/attendance)
- スケジュール (/schedule)
- タスク管理 (/tasks)
- メモ (/memo)
- 知識向上 (/knowledge)
- 月次振り返り (/review)
- 組織図 (/org)
- 設定 (/settings)

## KPI・成功指標

| 指標 | 目標 |
|------|------|
| HKR（開通率） | 80%以上 |
| 月次開通件数 | チーム目標による |
| 日次入力率 | メンバー全員が毎日入力 |

## 業務ルール

### 業務月（25日区切り）
- **当月業務月** = 前月25日 〜 当月24日
- 例: 5月業務月 = 5/25〜6/24
- 25日以降の獲得 → 当月業務月にカウント
- 24日以前の獲得 → 前月業務月にカウント

### 開通日の決定ルール
| タイプ | 開通日 |
|--------|--------|
| So-net | 工事日（construction_date）|
| WiMAX直せち | 獲得日（date）|
| WiMAX後送り | 受取日（delivery_date）|

### HKR計算
```
HKR = 開通数 / 解除数 × 100（%）
目標: 80%以上
解除数が0のときは計算不可（null）
```
